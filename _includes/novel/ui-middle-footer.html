Debug & HowTo
推奨
　fontSize: 18
　Spread(見開き)
　ブラウザサイズ（良い感じに）
　※推奨以外でも見れますので環境に合せてください。

諸注意
　ページ数は fontSize と 見開きor半分サイズ の組み合わせで無限に変動する。
　[12 S] は Spread(見開き) の時に丁度良いフォントサイズを意味する。
　[18 S H] は Spread(見開き) と Half(半分) の時に丁度良い。

Mouse
　クリック（左側/右側）: ページ進め/戻り

Key
　左右キー: ページ進め/戻り
　上下キー: フォントサイズ上げ/下げ
　Lキー: ライトテーマ
　Dキー: ダークテーマ
　Mキー: 横長サイズ
　Sキー: 見開きサイズ
　Hキー: 半分サイズ（スマホなどで？）
　Fキー: 集中モード

例
　dark にして multi にして 集中モード ( Dキー > Mキー > Fキー )
